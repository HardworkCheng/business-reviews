2. 了解 **数据库设计.md**
3. 查看 **前端接口文档.md** 和 **后端接口文档.md**

2. 参考接口文档进行开发
3. 遇到问题查看问题排查文档

1. 阅读 **阿里云OSS配置指南.md**
2. 执行数据库脚本
3. 配置环境变量
4. 启动服务

## 📝 文档维护说明

### 文档更新规则
1. 每次新增功能后，更新对应的功能文档
2. 修复问题后，添加问题修复说明
3. 接口变更后，及时更新接口文档
4. 数据库变更后，更新数据库设计文档

### 文档命名规范
- 功能文档：`功能名称_说明.md`
- 问题文档：`问题描述_修复说明.md`
- 接口文档：`模块名称_接口文档.md`
- 设计文档：`功能名称_设计方案.md`
