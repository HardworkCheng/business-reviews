# 商家运营中心评论管理页面UI重设计需求文档

## 简介

本需求旨在重新设计商家运营中心的评论管理页面，参照提供的HTML设计稿，实现现代化、数据可视化的评论管理界面，提升商家管理评论的效率和体验。

## 术语表

- **商家运营中心 (Merchant Center)**: 商家管理店铺、评论、优惠券等业务的后台管理系统
- **评论管理页面 (Comment Management Page)**: 商家查看和管理用户评论的页面
- **数据概览看板 (Dashboard Cards)**: 展示关键数据指标的卡片组件
- **评论状态 (Comment Status)**: 评论的显示状态，包括正常显示、已删除等
- **商家回复 (Merchant Reply)**: 商家对用户评论的回复内容

## 需求

### 需求 1: 数据概览看板

**用户故事:** 作为商家，我希望在评论管理页面顶部看到关键数据指标，以便快速了解评论整体情况。

#### 验收标准

1. WHEN 商家访问评论管理页面 THEN 系统应在页面顶部展示4个数据卡片
2. WHEN 展示数据卡片 THEN 系统应包含以下指标：今日新增评论数、店铺综合评分、待回复内容数、差评/投诉待处理数
3. WHEN 展示每个数据卡片 THEN 系统应显示标题、数值、趋势指标和图标
4. WHEN 用户悬停在数据卡片上 THEN 系统应展示卡片上浮和阴影加深的动画效果
5. WHEN 展示趋势指标 THEN 系统应使用不同颜色表示上升(绿色)、下降(红色)或持平(灰色)

### 需求 2: Tab分类筛选

**用户故事:** 作为商家，我希望通过Tab快速切换不同状态的评论，以便分类管理评论。

#### 验收标准

1. WHEN 商家查看评论列表 THEN 系统应提供Tab切换功能，包括"全部评论"、"正常显示"、"差评/投诉"、"已删除"
2. WHEN 展示每个Tab标签 THEN 系统应显示该分类的评论数量
3. WHEN 商家点击Tab THEN 系统应切换到对应分类的评论列表
4. WHEN 切换Tab THEN 系统应保持搜索条件和分页状态

### 需求 3: 增强的评论展示

**用户故事:** 作为商家，我希望看到详细的评论信息展示，以便更好地了解用户反馈。

#### 验收标准

1. WHEN 展示评论列表 THEN 系统应显示用户头像、用户名、VIP标识、匿名标识
2. WHEN 展示评论详情 THEN 系统应显示评分星级、评论时间、关联商品名称
3. WHEN 评论包含图片 THEN 系统应展示缩略图，并支持点击预览大图
4. WHEN 商家已回复评论 THEN 系统应在评论下方展示带有橙色背景的商家回复区域
5. WHEN 展示评论状态 THEN 系统应使用带颜色的状态点和文字标识

### 需求 4: 现代化操作按钮

**用户故事:** 作为商家，我希望使用直观的操作按钮管理评论，以便快速执行操作。

#### 验收标准

1. WHEN 展示评论操作区 THEN 系统应提供回复、置顶、删除三个圆形图标按钮
2. WHEN 用户悬停在操作按钮上 THEN 系统应显示按钮功能提示，并改变按钮背景颜色
3. WHEN 商家点击回复按钮 THEN 系统应打开回复对话框
4. WHEN 商家点击置顶按钮 THEN 系统应将评论置顶显示
5. WHEN 商家点击删除按钮 THEN 系统应弹出确认对话框，确认后删除评论

### 需求 5: 搜索和导出功能

**用户故事:** 作为商家，我希望搜索评论并导出数据，以便进行数据分析和备份。

#### 验收标准

1. WHEN 商家使用搜索功能 THEN 系统应支持按评论内容、用户名、订单号搜索
2. WHEN 商家输入搜索关键词 THEN 系统应实时过滤评论列表
3. WHEN 商家点击导出按钮 THEN 系统应导出当前筛选条件下的评论数据为Excel文件
4. WHEN 导出数据 THEN 系统应包含用户信息、评论内容、评分、时间、状态等字段

### 需求 6: 响应式设计和动画效果

**用户故事:** 作为商家，我希望页面具有流畅的交互体验，以便提升使用舒适度。

#### 验收标准

1. WHEN 页面加载 THEN 系统应使用平滑的过渡动画展示内容
2. WHEN 用户悬停在表格行上 THEN 系统应改变行的背景颜色
3. WHEN 用户悬停在数据卡片上 THEN 系统应展示卡片上浮效果
4. WHEN 页面元素交互 THEN 系统应使用贝塞尔曲线缓动函数实现流畅动画
5. WHEN 在不同屏幕尺寸下查看 THEN 系统应保持良好的布局和可读性

### 需求 7: 分页功能

**用户故事:** 作为商家，我希望通过分页浏览大量评论，以便高效管理评论数据。

#### 验收标准

1. WHEN 评论数量超过单页显示数量 THEN 系统应在页面底部展示分页组件
2. WHEN 展示分页组件 THEN 系统应显示总数、每页数量选择器、页码、跳转输入框
3. WHEN 商家切换页码 THEN 系统应加载对应页的评论数据
4. WHEN 商家改变每页数量 THEN 系统应重新加载评论列表并重置到第一页
5. WHEN 分页加载数据 THEN 系统应显示加载状态指示器
