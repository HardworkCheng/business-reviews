# 美食点评 uni-app 移动端应用

一个基于 uni-app + Vue3 的移动端美食点评应用，采用粘土乐园风格设计，提供美食推荐、店铺详情、笔记分享等功能。

## 项目简介

这是一个使用 uni-app 框架开发的跨平台移动端美食点评应用，包含以下核心功能：
- 🔐 登录页：手机号验证码登录、第三方登录（主页）
- 🏠 首页：美食推荐、分类浏览
- 📝 笔记详情：查看他人分享的美食探店笔记、点赞评论
- 🔍 搜索：搜索店铺和美食、筛选功能
- 🏪 店铺详情：查看店铺信息、用户评价
- ✍️ 发布笔记：分享自己的美食体验、上传图片
- 👤 个人主页：管理个人笔记和收藏
- 💬 消息：聊天和通知系统
- 🗺️ 地图：查看附近店铺位置、导航功能

## 技术栈

- **uni-app**：跨平台开发框架
- **Vue 3**：渐进式JavaScript框架（使用 Composition API）
- **SCSS**：CSS预处理器
- **uni-ui**：uni-app官方组件库
- **支持平台**：微信小程序、H5、App

## 功能说明

### 1. 首页 (index.html)
- 顶部定位和搜索
- 8个主要分类图标（美食、KTV、丽人·美发等）
- 瀑布流笔记推荐
- 底部导航栏

### 2. 笔记详情 (note-detail.html)
- 大图展示
- 作者信息和关注
- 笔记内容和标签
- 点赞、收藏、评论功能
- 评论列表和回复

### 3. 搜索结果 (search.html)
- 顶部搜索栏
- 筛选条件（距离、人气、评分、价格）
- 商家列表展示
- 点击跳转店铺详情

### 4. 店铺详情 (shop-detail.html)
- 店铺图片和基本信息
- 评分细项（口味、环境、服务）
- 地址、营业时间、优惠信息
- 用户评价标签
- 点餐和电话按钮
- 用户评价列表

### 5. 发布笔记 (publish.html)
- 照片上传
- 标题和内容编辑
- 关联商户
- 添加话题和位置
- 字数统计
- 发布/取消操作

### 6. 个人主页 (profile.html)
- 个人信息展示
- 关注/粉丝/获赞/收藏数据
- Tab切换（我的笔记、收藏、足迹）
- 笔记网格展示

### 7. 消息 (message.html)
- 聊天/通知Tab切换
- 聊天列表（在线状态、未读数）
- 通知列表（点赞、评论、关注）

### 8. 地图 (map.html)
- 模拟地图展示
- 店铺标记点
- 分类筛选
- 店铺信息卡片
- 导航功能

### 9. 登录 (login.html)
- 手机号验证码登录
- 第三方登录（微信、QQ、微博）
- 游客模式
- 用户协议

## 页面跳转逻辑

```
登录页 (login.html)
    ↓
首页 (index.html)
    ├→ 搜索 → 搜索结果 (search.html) → 店铺详情 (shop-detail.html)
    ├→ 笔记卡片 → 笔记详情 (note-detail.html)
    ├→ 地图 (map.html) → 店铺详情 (shop-detail.html)
    ├→ 发布 (publish.html)
    ├→ 消息 (message.html)
    └→ 个人 (profile.html) → 笔记详情 (note-detail.html)
```

## 快速开始

### 环境要求

- Node.js 14+
- HBuilderX（推荐）或 微信开发者工具
- npm 或 yarn

### 安装依赖

```bash
# 进入项目目录
cd front-business-reviews-Mobile

# 安装依赖
npm install
# 或
yarn install
```

### 运行项目

#### 方法一：使用 HBuilderX（推荐）

1. 用 HBuilderX 打开项目
2. 点击菜单栏 运行 -> 运行到内置浏览器
3. 或运行到微信小程序开发者工具

#### 方法二：命令行运行

```bash
# 运行到 H5
npm run dev:h5

# 运行到微信小程序
npm run dev:mp-weixin

# 运行到 App
npm run dev:app
```

### 编译发布

```bash
# 编译为 H5
npm run build:h5

# 编译为微信小程序
npm run build:mp-weixin

# 编译为 App
npm run build:app
```

## 开发说明

### 样式规范

- 使用 SCSS 预处理器
- 尺寸单位使用 rpx（响应式像素）
- 颜色变量统一在 `static/styles/common.scss` 中定义
- 粘土风格样式类名以 `clay-` 开头

### 组件规范

- 使用 Vue 3 Composition API
- 页面使用 `<script setup>` 语法
- 统一使用 uni-app 生命周期 hooks
