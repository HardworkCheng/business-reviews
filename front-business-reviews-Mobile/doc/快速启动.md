### 🚀 如何运行

#### 使用 HBuilderX（推荐）

1. **安装 HBuilderX**
   - 下载地址：https://www.dcloud.io/hbuilderx.html
   - 安装后打开 HBuilderX

2. **导入项目**
   - 文件 -> 导入 -> 从本地目录导入
   - 选择 `front-business-reviews-Mobile` 目录

3. **运行项目**
   - **运行到浏览器**：运行 -> 运行到浏览器 -> Chrome
   - **运行到微信小程序**：运行 -> 运行到小程序模拟器 -> 微信开发者工具
   - **运行到手机**：运行 -> 运行到手机或模拟器

#### 使用命令行

1. **安装依赖**
```bash
cd front-business-reviews-Mobile
npm install
```

2. **运行到H5**
```bash
npm run dev:h5
```
访问 http://localhost:5173

3. **运行到微信小程序**
```bash
npm run dev:mp-weixin
```
然后用微信开发者工具打开 `unpackage/dist/dev/mp-weixin` 目录

### 📝 技术要点

1. **Vue3 Composition API**
   - 所有页面使用 `<script setup>` 语法
   - 使用 ref 和 reactive 管理状态

2. **uni-app API**
   - 导航：uni.navigateTo、uni.switchTab、uni.navigateBack
   - 交互：uni.showToast、uni.showModal
   - 选择：uni.chooseImage
   - 定位：uni.getLocation

3. **响应式单位**
   - 使用 rpx 作为尺寸单位（750rpx = 屏幕宽度）
   - 1rpx = 屏幕宽度 / 750

### 🔧 常见问题

**Q1：TabBar 图标不显示怎么办？**
A：
- 已为您创建了所有 TabBar 图标（SVG 格式）
- 图标位于：`static/tabbar/` 目录
- 图标设计参考了 dianping 项目中的 FontAwesome 图标
- 如果 HBuilderX 不支持 SVG，可以将图标转换为 PNG 格式

**Q2：如何修改TabBar图标？**
A：在 `static/tabbar/` 目录下替换图标文件，然后在 pages.json 中配置路径

**Q3：如何添加新页面？**
A： 
1. 在 pages 目录下创建新的页面目录和 .vue 文件
2. 在 pages.json 中添加页面配置

**Q4：如何配置微信小程序？**
A：在 manifest.json 的 mp-weixin 节点中配置 appid

**Q5：图片资源在哪里？**
A: 
- 项目图片：放在 static 目录
- 可以使用 dianping/assets/img 中的图片
