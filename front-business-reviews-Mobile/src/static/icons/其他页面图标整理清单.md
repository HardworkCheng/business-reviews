# 其他页面图标整理清单

## 笔记详情页图标（已配置）

笔记详情页需要以下6个PNG图标，请将对应图标放入 `src/static/icons/` 目录：

### 点赞图标
| 图标文件名 | 说明 | 建议尺寸 |
|-----------|------|---------|
| `like.png` | 未点赞状态 | 80x80px |
| `like-active.png` | 已点赞状态 | 80x80px |

### 收藏图标
| 图标文件名 | 说明 | 建议尺寸 |
|-----------|------|---------|
| `bookmark.png` | 未收藏状态 | 80x80px |
| `bookmark-active.png` | 已收藏状态 | 80x80px |

### 关注图标
| 图标文件名 | 说明 | 建议尺寸 |
|-----------|------|---------|
| `follow.png` | 未关注状态（+号） | 70x70px |
| `follow-active.png` | 已关注状态（✓号） | 70x70px |

### 业务逻辑说明
- 当用户查看自己发布的笔记时，**不显示关注按钮**
- 点赞和收藏按钮始终显示
- 图标会根据状态自动切换（已点赞/未点赞、已收藏/未收藏、已关注/未关注）

## 当前状态分析

经过详细检查，发现其他页面主要使用以下几种图标方式：

### 1. 已使用 Unicode 字符的页面（建议保持）
这些页面使用简单的 Unicode 字符，显示效果良好，建议保持现状：

#### 导航返回按钮
- **字符**：`←`
- **使用页面**：user-profile.vue, user-list.vue, shop-detail.vue, settings.vue
- **建议**：保持使用 Unicode 字符

#### 功能按钮
- **分享按钮**：`📤` (shop-detail.vue)
- **喜欢按钮**：`❤️` (shop-detail.vue, user-profile.vue)
- **相机按钮**：`📷` (publish.vue, settings.vue)
- **电话按钮**：`📞` (shop-detail.vue)
- **餐厅按钮**：`🍽️` (shop-detail.vue)

4. **heart.png** - 喜欢图标（普通状态）

## 修改建议

### 方案1：保持现状（推荐）
- 优点：简单、兼容性好、文件小
- 缺点：视觉效果相对简单
- 适合：快速开发、简约风格

### 方案2：部分替换为 PNG 图标
只替换关键功能的图标，保持其他 Unicode 字符：

**优先替换列表**：
1. heart.png / heart-filled.png（喜欢功能）
2. camera.png（相机功能）
3. share.png（分享功能）
4. back.png（返回按钮）

### 方案3：全面使用 PNG 图标
将所有 Unicode 字符替换为 PNG 图标，获得最佳视觉效果。

## 如果选择添加 PNG 图标

### 需要修改的文件

#### 1. shop-detail.vue
```html
<!-- 当前 -->
<text>←</text>
<text>📤</text>
<text>❤️</text>
<text>📞</text>

<!-- 修改为 -->
<image src="/static/icons/back.png" class="icon-img" mode="aspectFit"></image>
<image src="/static/icons/share.png" class="icon-img" mode="aspectFit"></image>
<image src="/static/icons/heart.png" class="icon-img" mode="aspectFit"></image>
<image src="/static/icons/phone.png" class="icon-img" mode="aspectFit"></image>
```

#### 2. publish.vue
```html
<!-- 当前 -->
<text class="add-icon">+</text>
<text>📷</text>
<view class="delete-btn" @click="removeImage(index)">×</view>

<!-- 修改为 -->
<image src="/static/icons/add.png" class="icon-img" mode="aspectFit"></image>
<image src="/static/icons/camera.png" class="icon-img" mode="aspectFit"></image>
<image src="/static/icons/delete.png" class="icon-img" mode="aspectFit"></image>
```

#### 3. settings.vue
```html
<!-- 当前 -->
<text class="icon">←</text>
<text>📷 点击修改</text>

<!-- 修改为 -->
<image src="/static/icons/back.png" class="icon-img" mode="aspectFit"></image>
<image src="/static/icons/camera.png" class="icon-img" mode="aspectFit"></image>
```

### CSS 样式添加
```scss
.icon-img {
  width: 32rpx;
  height: 32rpx;
}

.icon-img-small {
  width: 24rpx;
  height: 24rpx;
}

.icon-img-large {
  width: 48rpx;
  height: 48rpx;
}
```

## 图标规格要求

- **尺寸**：32x32 像素（常规）、24x24 像素（小图标）
- **格式**：PNG，透明背景
- **颜色**：
  - 普通状态：#666666
  - 激活状态：#FF9E64
  - 危险操作：#FF4444
  - 成功状态：#06D6A0

## 总结

**推荐方案**：保持现状，因为：
1. 当前的 Unicode 字符显示效果已经很好
2. 文件体积小，加载快
3. 兼容性好，不会出现图片加载失败的问题
4. 开发和维护成本低

如果你希望提升视觉效果，可以选择性地替换几个关键图标（如喜欢、分享、相机），其他保持 Unicode 字符。
