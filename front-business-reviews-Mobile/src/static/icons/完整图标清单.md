## 图标规格要求
- **尺寸**：建议 48x48 像素或 64x64 像素
- **格式**：PNG 格式，支持透明背景
- **颜色**：
  - 普通状态：#999999（灰色）
  - 激活状态：#FF9E64（橙色）
  - add 图标特殊：始终使用橙色，激活状态更大更粗

## 使用的文件位置

### pages.json（底部导航栏）
```json
"iconPath": "static/icons/home.png",
"selectedIconPath": "static/icons/home-active.png"
```

### pages/index/index.vue（首页）
```javascript
// 搜索和箭头图标
<image src="/static/icons/arrow-down.png" />
<image src="/static/icons/search.png" />

// 分类图标
{ name: '美食', icon: '/static/icons/food.png' }
```

### pages/search/search.vue（搜索页）
```javascript
// 分类图标
{ name: '美食', icon: '/static/icons/food.png', id: 1 }
```

## 替换方法
1. 准备好对应的 PNG 图标文件
2. 确保文件名与上述列表完全一致
3. 直接替换 `src/static/icons/` 目录中的同名文件
4. 重新编译运行项目即可看到新图标

## 注意事项
- 文件名必须完全匹配（区分大小写）
- 必须是 PNG 格式（uniapp tabBar 不支持 SVG）
- 建议保持透明背景
- 图标尺寸建议统一，避免显示异常
