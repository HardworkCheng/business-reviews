# 临时图标解决方案

## 当前问题
uniapp tabBar 不支持 SVG 格式，导致底部导航图标无法正常显示。

## 临时解决方案
由于无法直接创建 PNG 图片文件，建议以下几种解决方案：

### 方案1：使用 iconfont 字体图标
1. 去 iconfont.cn 下载对应的字体图标
2. 在项目中引入字体文件
3. 使用 text 组件显示图标

### 方案2：使用在线工具转换
1. 将现有的 SVG 文件复制到在线 SVG 转 PNG 工具
2. 转换为 48x48 或 64x64 的 PNG 文件
3. 下载并替换到 icons 目录

### 方案3：使用设计软件
1. 用 Photoshop、Figma 等工具打开 SVG 文件
2. 导出为 PNG 格式
3. 保存到 icons 目录

## 推荐的在线转换工具
- https://convertio.co/svg-png/
- https://cloudconvert.com/svg-to-png
- https://www.aconvert.com/image/svg-to-png/
