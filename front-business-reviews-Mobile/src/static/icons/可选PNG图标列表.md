# 可选 PNG 图标列表

## 当前状态
✅ 主要功能图标已完成（底部导航、搜索、分类等）
✅ 其他页面主要使用 Unicode 字符，显示正常

## 可选添加的 PNG 图标

如果你想进一步提升视觉效果，可以考虑添加以下图标：

### 高优先级（推荐添加）
1. **heart.png** - 喜欢图标（空心，普通状态）
2. **heart-filled.png** - 喜欢图标（实心，已喜欢状态）
3. **camera.png** - 相机图标
4. **share.png** - 分享图标

### 中优先级
5. **back.png** - 返回箭头图标
6. **phone.png** - 电话图标
7. **edit.png** - 编辑图标

### 低优先级
8. **close.png** - 关闭 X 图标
9. **add-small.png** - 小尺寸加号图标
10. **delete.png** - 删除图标

## 图标规格
- **尺寸**：32x32 像素
- **格式**：PNG，透明背景
- **颜色**：#666666（普通状态）

## 使用方法

### 1. 准备图标文件
将对应的 PNG 文件放入 `src/static/icons/` 目录

### 2. 修改代码（可选）
如果要替换 Unicode 字符，需要修改对应页面的代码

### 3. 示例修改
```html
<!-- 替换前 -->
<text>❤️</text>

<!-- 替换后 -->
<image src="/static/icons/heart.png" class="icon-img" mode="aspectFit"></image>
```

## 建议
**当前的 Unicode 字符已经能很好地满足需求，建议保持现状。**

如果你想要更精致的视觉效果，可以优先添加前4个高优先级图标。
