# 商家运营中心店铺信息页面 UI 重新设计

## 更新日期
2025-12-22

## 设计参考
参考文件：`商家运营中心.html`

## 设计理念

采用经典的后台管理系统设计风格，强调：
- **简洁专业**：白色卡片 + 浅灰背景
- **清晰布局**：左右分栏，信息层次分明
- **品牌色彩**：橙色 (#FF6B00) 作为主题色
- **高效操作**：表单字段紧凑排列，提升填写效率

## 主要变更

### 1. 整体布局

#### 之前：
- 垂直布局：封面在上，表单在下
- 表单字段分散在多个区域
- 使用了多个分隔线和盒子

#### 现在：
- **水平布局**：左侧封面 (280px) + 右侧表单 (flex)
- **2列网格**：表单字段采用2列网格布局
- **统一卡片**：所有内容在一个白色卡片中

### 2. 配色方案

| 元素 | 之前 | 现在 |
|------|------|------|
| 页面背景 | 白色 (#FFFFFF) | 浅灰 (#f0f2f5) |
| 卡片背景 | 白色 + 阴影 | 白色 + 轻阴影 |
| 主题色 | 橙色 (#FF7D00) | 橙色 (#FF6B00) |
| 边框 | 灰色 (#E5E5E5) | 灰色 (#d9d9d9) |
| 文字 | 深灰 (#171717) | 深灰 (#333) |

## 视觉对比

### 之前的设计特点
- 渐变背景和装饰性元素较多
- 圆角较大，视觉较柔和
- 字体较大，间距较宽松
- 适合展示型页面

### 现在的设计特点
- 简洁专业的后台风格
- 直角设计，视觉更清晰
- 字体适中，信息密度更高
- 适合高效操作的管理页面
